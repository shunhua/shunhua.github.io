有赞vantUI学习 step1

1、安装淘宝镜像

npm install -g cnpm –registry=https://registry.npm.taobao.org

2、安装vue-cli

cnpm install –global vue-cli

3、使用vue-cli安装项目[bsh]

vue init youzan/vue-cli-template-vant bsh

4、安装vantUI

cnpm i vant -S

5、引入组件 推荐使用安装 babel-plugin-import 插件

npm i babel-plugin-import -D

6、install dependencies 安装依赖

npm install

7、# serve with hot reload at localhost:8080

npm run dev
