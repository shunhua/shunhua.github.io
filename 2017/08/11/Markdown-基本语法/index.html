Markdown 基本语法

2017-08-11

^本文主要介绍markdown

Markdown 基本语法——

Ⅰ. 标题设置（让字体变大，和word的标题意思一样）
在Markdown当中设置标题，有两种方式：
a：通过在文字下方添加“等号”和“减”，他们分别表示一级标题和二级标题。
b：在文字开头加上 “星”，通过“星”数量表示几级标题。（一共只有1~6级标题，1级标题字体最大）
example:
a、
This is an H1
====
This is an H2

b、

This is an H1

This is an H2

This is an H3

This is an H4

This is an H5
This is an H6

Ⅱ. 块注释（blockquote）
通过在文字开头添加“>”表示块注释。（当>和文字之间添加五个blank时，块注释的文字会有变化。）
example:

简单引用1
简单引用2

多行引用

嵌套引用

引用中使用Markdown语法。

  • 这是第一行列表项。
  • 这是第二行列表项。

给出一些例子代码：

return shell_exec("echo $input | $markdown_script");

Ⅲ. 斜体
将需要设置为斜体的文字两端使用1个“星号”或者“下划线”夹起来
example:

a、星号、 斜体

b、下划线、 斜体

Ⅳ. 粗体
将需要设置为斜体的文字两端使用2个“星号”或者“下划线”夹起来
example:

a、星号、 斜体
b、下划线、 斜体

Ⅴ. 无序列表
在文字开头添加(星, 加, and 减)实现无序列表。但是要注意在(星, 加, and 减)和文字之间需要添加空格。（建议：一个文档中只是用一种无序列表的表示方式）

example:

  • kobe
  • bryant
  • lakers

Ⅵ. 有序列表
使用数字后面跟上句号。（还要有空格）

example:

  1. Bird
  2. McHale

Ⅶ. 链接（Links）
Markdown中有两种方式，实现链接，分别为内联方式和引用方式。

  • 行内式
    This is baidu
    baidu
  • 参考式
    This is [baidu example][id] reference-style link.
    标记: [id]: https://www.baidu.com/ “度娘”
    或者: [id]: https://www.baidu.com/ ‘度娘’ (简书不支持)
    或者 [id]: https://www.baidu.com/ (度娘)
