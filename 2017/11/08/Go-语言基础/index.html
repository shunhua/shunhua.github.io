  • defer
    defer 语句会延迟函数的执行直到上层函数返回。
    延迟调用的参数会立刻生成，但是在上层函数返回前函数都不会被调用。

复杂类型

  • 指针
    Go 具有指针。 指针保存了变量的内存地址

  • 结构体
    一个结构体（struct）就是一个字段的集合

  • 数组
    类型 [n]T 是一个有 n 个类型为 T 的值的数组

  • slice
    一个 slice 会指向一个序列的值，并且包含了长度信息
    s[lo:hi]表示从 lo 到 hi-1 的 slice 元素，含两端。因此
    s[lo:lo]是空的，而
    s[lo:lo+1]有一个元素。
