搭建一套规范的 Vue3.x 项目工程环境

2021-11-04

一、使用 Vite 快速初始化项目雏形

npm init @vitejs/app

  • 输入项目名称
  • 选择模板 vue-ts
  • npm install
  • npm run dev

修改 Vite 配置文件 vite.config.ts

规范目录结构

二、集成路由工具 Vue Router

npm i vue-router@4

三、集成状态管理工具 Vuex

npm i vuex@next

四、集成 HTTP 工具 Axios

npm i axios

使用 axios

至此，一个基于 TypeScript + Vite + Vue3 + Vue Router + Vuex + Axios + Stylus/Sass/Less 的前端项目开发环境搭建完毕。
