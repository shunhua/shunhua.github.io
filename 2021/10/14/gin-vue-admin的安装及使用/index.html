gin-vue-admin的安装及使用

2021-10-14

官网地址： [ gin-vue-admin官网 ]
gitHub： [ gin-vue-admin ]

  • 克隆项目

    git clone git@github.com:flipped-aurora/gin-vue-admin.git

  • 进入项目gin目录

    cd gin-vue-admin/server
    go run main.go

  • 进入项目vue目录

    cd gin-vue-admin/web
    npm i
    npm run serve

  • 打开web网站 开始初始化
