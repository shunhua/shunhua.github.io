linux 常用小技巧

2022-01-12

查看历史记录:

Ctrl + R ： 用于查找使用过的命令

例如，我查找 docker 这个命令，就可以用 Ctrl + R，按回车，然后只输入 doc 即可。

history 命令

可以在 .bash_history 文件中查看所有 history 命令记录。

history -c

执行的命令前，加一个空格，那这条命令就不会被 history 保存到历史记录

追踪及调试:

strace + 常用命令

  • -f 跟踪由fork调用所产生的子进程.
  • -c 统计每一系统调用的所执行的时间,次数和出错的次数等.
  • -p pid 跟踪指定的进程pid.

gdb + 常用命令

  • -p pid 跟踪指定的进程pid.

查看基本信息:

lscpu cpu的相关信息

hostnamectl 虚拟机信息

cal 查看日历

快速清空文件:

$ > xxx.log

动态实时查看日志:

tail -f xxx.log
